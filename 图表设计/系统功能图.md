## 四、系统功能流程图

### 4.1 学分申领流程

```mermaid
graph LR
    A[学生] --> B[登录系统]
    B --> C[进入学分申领页面]
    C --> D[上传获奖证明图片]
    D --> E[填写获奖名称]
    E --> F[选择所属学年]
    F --> G[提交申请]
    G --> H{管理员审核}
    H -->|批准| I[状态:已批准]
    H -->|驳回| J[状态:已驳回]
    I --> K[学分计入总数]
    J --> L[学生查看驳回原因]
    L --> M[修改后重新提交]
    M --> H
    
    classDef user fill:#3498DB,stroke:#2E86C1,stroke-width:2px,color:white
    classDef process fill:#F39C12,stroke:#D68910,stroke-width:2px,color:black
    classDef decision fill:#E74C3C,stroke:#C0392B,stroke-width:2px,color:white
    classDef result fill:#58D68D,stroke:#28B463,stroke-width:2px,color:black
    
    class A user
    class B,C,D,E,F,G process
    class H decision
    class I,J,K,L,M result
```

```mermaid
graph LR
```

### 4.3 用户认证流程

```mermaid
graph LR
    A[访问系统] --> B{是否已登录}
    B -->|否| C[跳转登录页]
    C --> D[输入用户名密码]
    D --> E{验证通过}
    E -->|是| F[生成JWT Token]
    F --> G[保存Token到本地]
    G --> H[进入系统首页]
    E -->|否| I[提示错误信息]
    I --> D
    
    B -->|是| H
    
    H --> I2[请求携带Token]
    I2 --> J{网关鉴权}
    J -->|通过| K[访问业务接口]
    J -->|失败| L[跳转登录页]
    
    classDef start fill:#3498DB,stroke:#2E86C1,stroke-width:2px,color:white
    classDef process fill:#F39C12,stroke:#D68910,stroke-width:2px,color:black
    classDef decision fill:#E74C3C,stroke:#C0392B,stroke-width:2px,color:white
    classDef success fill:#58D68D,stroke:#28B463,stroke-width:2px,color:black
    
    class A start
    class B,E,J decision
    class C,D,F,G,H,I2,K process
    class I,L process
```
